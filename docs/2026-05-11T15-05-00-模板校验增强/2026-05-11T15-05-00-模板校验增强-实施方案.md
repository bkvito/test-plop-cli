# 模板校验增强实施方案

## 目标

- 将模板自动校验范围从 `src/index.vue` 扩展到多个生成文件
- 为不同业务模板补充更有辨识度的特征断言
- 优化失败提示，缩短模板问题定位时间

## 实施步骤

1. 重构 `scripts/verify-templates.mjs` 的规则结构
2. 将断言从“按模板组合的单列表”改成“按文件路径分组”
3. 为 `general`、`single-pick`、`multiple-pick` 分别补充差异化断言
4. 重新执行全量验证命令确认脚本稳定

## 伪代码

```js
for each 模板组合:
  生成模板输出
  for each 需要校验的文件:
    先检查文件是否存在
    再检查文件中是否包含关键片段
```

## 风险控制

- 只增强校验脚本，不修改生成核心逻辑
- 断言选择关键片段而非整文件比对，降低误报概率
- 验证产物继续限制在 `tmp-output/template-verify`
